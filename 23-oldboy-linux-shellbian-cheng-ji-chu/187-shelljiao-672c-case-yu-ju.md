#### 1、case结构条件句语法

```
case "字符串变量" in
    值1）
        指令1
        ;;
    值2）
        指令2
        ;;
    *)
        指令
esac
```

注意：case语句相当于一个if的多分支结构语句

```
值1的选项
apple）
    echo -e "@RED_COLOR apple $RES"
    ;;
也可以这样写，输入2种格式找同一个选项
apple|APPLE)
    echo -e "$RED_COLOR apple $RES"
    ;;
```

##### case 语句小结

1）case语句就相当于多分支的if语句。case语句的优势是更规范，易读。

2）case语句适合变量的值少，且为固定的数字或字符串集合。

3）系统服务启动脚本传参的判断多用case语句
